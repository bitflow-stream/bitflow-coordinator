Bitflow Backend API

Base URL: /bitflow, Version: 1.0.0, Find out more about Swagger

This file defines all possible communication between bitflow frontend and backend.

Schemes: http

Summary

Tag: project

Operations about projects

Operation Description
GET /projects

Query all projects assigned to the requesting user.
POST /project

Create a new project.
GET /project/{id}

Query information about the specified project.
POST /project/{id}

Update an existing project.
DELETE /project/{id}

Delete specified project.
GET /project/{id}/users

Query all users with access to the specified project.
DELETE /project/{projectId}/users/{userId}

Removing a user from the specified project.
POST /project/{projectId}/users/{userId}

Adding a user to the specified project.
